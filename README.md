# RushPirateTower

A third-person action defense game where the player protects a pirate tower from waves of rushing skeleton enemies.

## Project overview
- Engine: Unreal Engine 5.6
- Project type: Blueprint-based (Third Person template)
- Default map: /Game/ThirdPerson/Lvl_ThirdPerson
- Target hardware class: Mobile (scalable)
- Input system: Enhanced Input + mobile touch interface

## Core gameplay
- Defend the pirate tower against incoming skeleton waves
- Third-person combat focused on aim and positioning
